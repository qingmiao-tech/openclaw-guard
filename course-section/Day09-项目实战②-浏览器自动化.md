## 第 4 页 · Browser Relay 架构

### 三层桥接

```
OpenClaw Gateway → Node 节点 → Chrome Browser Relay 扩展
```

### 工作原理

1. Gateway 发送操作指令
2. Node 节点中转指令
3. Chrome 扩展在浏览器中执行操作

### 支持的操作

- 🌐 打开页面
- 🖱️ 点击元素
- ⌨️ 输入文本
- 📋 提取内容
- 📸 截图

## 第 5 页 · 安装步骤

### 第一步：安装 OpenClaw Node

```bash
# 安装 Node 服务
openclaw node install

# 启动 Node 节点
openclaw node restart
```

### 第二步：安装 Chrome 扩展

1. `openclaw browser extension install` → 获取扩展文件
2. 打开 Chrome → `chrome://extensions`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择扩展文件夹

### 第三步：确认连接

- 扩展状态显示 **绿色 Ready** ✅

## 第 6 页 · 快速测试

### 验证 AI 能控制浏览器

## 第 11 页 · 故障排查速查

| 问题 | 解决方案 |
|------|----------|
| 扩展状态不是 Ready | 确认 Node 节点已启动；刷新扩展页面 |
| AI 说无法访问浏览器 | 确认 Node 节点在线；检查连接状态 |
| AI 无法读取页面内容 | 确认扩展已 attach 到当前页面；刷新重试 |
| 操作超时 | 检查网络速度；简化操作指令 |

---

## 第 12 页 · 🔒 安全提示

- **浏览器控制权限**：不要在包含敏感信息的页面（如网银）上启用 Browser Relay
- **避免自动登录敏感账户**：不要让 AI 操作涉及资金、密码的页面
- **注意隐私**：AI 可以读取页面内容，避免在含有隐私信息的页面上使用
