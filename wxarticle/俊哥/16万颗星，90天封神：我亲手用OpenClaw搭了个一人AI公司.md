然后我打开 Telegram，跟我的开发助理说：

"帮我做两个宝宝起名网站，一个中国风的，古典大气那种，一个二次元可爱风的，萌萌的那种。做完直接部署到 Cloudflare Pages 上。"

发完消息，我去倒了杯水。

在手机上打开了Windows的记事本。。。

好了，接下来这个案例，是真正让我觉得"未来已来"的时刻。

我的部署方案是这样的：

腾讯云服务器跑 OpenClaw 的 Gateway（网关），我自己的 Windows 电脑跑 Node（节点），中间用 SSH 隧道打通。

这意味着什么呢？

意味着我的 AI 既有云端的稳定性，又能操控我本地的 Windows 电脑。

**混合部署，云端+本地，两手都要硬。**

这玩意到底是什么？

说了这么多我自己的体验，来正经聊聊 OpenClaw 本身。

一句话总结：

**OpenClaw 是一个 AI 调度器。它不生产智能，它调度智能。**

你可以把它理解成一个超级路由器，连接四样东西：

**第一层，连大脑。** Claude、GPT-5.3、Gemini、Grok、GLM-5、Kimi。。。市面上能叫得上名字的大模型，它基本都支持。你想用哪个用哪个，随便切。

**第二层，连耳朵和嘴巴。** Telegram、WhatsApp、Discord、Slack、iMessage、飞书、企业微信、QQ。。。12 个以上的消息平台。

对了说到这个，我现在同时接了四个渠道——**Telegram、QQ、企业微信、飞书**。

四个平台，同一个 AI 大脑。

我在 Telegram 上跟它聊完的事情，切到企业微信它还记得。

跨平台记忆，这个真的太香了。

**第三层，连手脚。** 浏览网页、管理邮件、运行脚本、控制智能家居、管理文件系统。就是我前面说的，能帮你打开记事本、读文件那种能力。

**第四层，连记忆。** 跨会话的长期记忆，存储为本地 Markdown 文件。它不会每次聊天都失忆，它记得你是谁，记得你之前说过什么。

硬件要求呢？

2 核 CPU，2GB 内存。

就这？

**就这。**

你家吃灰的旧电脑可能都能跑。

龙虾蜕壳：一个名字的三次冒险

这段故事真的太好笑了，必须讲。

OpenClaw 最早不叫 OpenClaw。

最早叫 **Clawdbot**，取 Claude + Claw 的谐音。挺聪明的对吧？

首周就拿下 6.8 万 GitHub 星标。

然后 Anthropic 来了。

"你这个名字跟我们的 Claude 太像了，商标侵权。"

创始人 Peter Steinberger 第二天就改名叫 Moltbot。

好了，故事到这里还算正常。

**接下来的事情才是真正的魔幻现实主义。**

改名后——注意是改名后——**10 秒内**。

加密货币骗子抢注了旧账号。

发行了虚假的 $CLAWD 代币。

市值冲到了 **1600 万美元**。

然后崩盘了。

哈哈哈哈哈哈。

Web3 的速度就是这么离谱。你前脚刚改名，后脚骗子就把你的旧名字做成了 meme 币。

两天后，项目最终定名 OpenClaw。

开源、大气、没有商标风险。

这段经历真的像龙虾蜕壳——疼，但蜕完之后更强了。

安全：必须聊，不能回避

好了，说点严肃的。

安全研究员提出了一个"致命三要素"：

1.能访问你的私有数据

2.能接触不可信内容

3.能对外通信

OpenClaw 三条全占。

通过 Shodan 扫描，已经发现了大约 **780 个暴露明文凭据的实例**。

这意味着什么？意味着有 780 个人的 AI 管家大门敞开，谁都能进去指挥。

吓人不？

吓人。

但我觉得，这不是 OpenClaw 的问题，这是**所有 AI Agent 框架**都要面对的问题。

你给了 AI 这么大的权限，安全就必须跟上。

我自己的做法是：独立机器跑 OpenClaw，跟日常环境物理隔离。SSH 隧道加密通信。权限最小化原则。

**能力越大，责任越大。这句话在 AI Agent 时代，比以往任何时候都真实。**

最后，五个判断

聊了这么多，我想认真说说我对这件事的看法。

**第一，AI Agent 时代已经到了。**

不是"即将到来"，不是"未来可期"。是已经到了。我每天都在用 6 个 AI Agent 干活，这不是 demo，这是我的日常。

**第二，开源正在吃掉封闭。**

OpenClaw 167,000 颗星不是因为营销做得好。是因为它真的开源，真的好用，真的能 self-host。你的数据在你自己手里，这在今天太稀缺了。

**第三，硬件文艺复兴正在发生。**

人们开始买 Mac Mini 来跑 AI Agent，开始出现 ClawBox 这样的专用硬件概念。AI 不只活在云端了，它开始住进你的房间。

**第四，安全是个未解难题，但不会阻止这波浪潮。**

就像互联网早期也有无数安全问题，但没有人因此不用互联网。该来的挡不住。

**第五，OpenClaw 正在成为 AI Agent 时代的 Linux。**

这不是夸张。Robert Scoble 说它是"自 Stable Diffusion 以来最重要的开源 AI 项目"。我个人觉得，它的意义甚至可能更大。
