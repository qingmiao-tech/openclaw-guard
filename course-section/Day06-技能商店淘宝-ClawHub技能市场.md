## 第 12 页 · 🔒 安全警告：ClawHub 供应链安全危机

### ⚠️ 真实事件：ClawHub 恶意技能大规模入侵

2026 年 2 月，多家安全机构（Koi Security、Snyk、Bitdefender）对 ClawHub 市场进行了全面审计，发现了严重的供应链安全问题：

| 发现 | 数据 |
|------|------|
| 恶意技能总数 | 1,184+ 个 |
| 占市场总量比例 | 约 7-20% |
| 最大攻击活动 | "ClawHavoc"——335 个技能伪装成正常工具，实际植入 macOS 信息窃取器 |
| 凭证泄露漏洞 | 283 个技能存在 API Key 和 SSH 密钥暴露风险 |
| 伪装重灾区 | 加密货币交易工具、生产力工具 |

### 恶意技能的三种常见手法

1. **偷凭证**：技能运行时悄悄将你的 API Key、SSH 密钥发送到攻击者服务器
2. **窃取文件**：处理文档时将文件上传到第三方
3. **伪装安装**：通过 fake pre-requisites 诱导安装恶意软件（如 Atomic Stealer）

### 安装前必做：五步安全审查

1. **先装 skill-vetting**：这是安全审查专用技能，每次安装新技能前先让 AI 审查
2. **检查权限要求**：一个"计算器"要求"完全文件系统访问"？🚩 红旗信号
3. **看下载量和星标**：优先选高下载量、GitHub 星标多的
4. **查看更新频率**：活跃维护的更可靠，长期无更新的要警惕
5. **阅读源码**：如果有 GitHub 链接，花 2 分钟扫一眼核心代码

> 🚨 ClawHub 目前没有官方审核机制，所有技能都是"自负风险"。安全意识是你最好的防火墙。

## 第 14 页 · 社区实战案例：Skills 安全避坑 + 25 个亲测技能

> 💬 来源：社区实战配置全记录

### 5000+ 技能怎么选？三步排雷法

在研究了 565 个 Skills 后，总结出一套安全筛选方法：

**第一步：先装 skill-vetting**

这是唯一建议第一个安装的技能——专门检查其他技能的安全性。装好后每次安装新技能前问 AI："帮我审查一下 [技能名] 的安全性。"

**第二步：看权限要求**

红旗信号：一个"计算器"技能要求"完全文件系统访问"——这不合理。

**第三步：优先选有口碑的**

GitHub 星标多、下载量高、社区推荐的，踩坑概率小。

### 社区已发现的安全隐患

- 偷 API Key：技能运行时悄悄把你的 Key 发到作者服务器
- 上传文件：处理文档时把文件传到第三方
- 植入后门：代码里藏命令注入

> ⚠️ 技能商店没有官方审核机制，所有技能都是"自负风险"。

### 25 个亲测技能清单（精选 5 个必装）

| 技能 | 功能 | 为什么必装 |
|------|------|-----------|
| skill-vetting | 技能安全扫描 | 安全第一，最先装 |
| agent-browser | 浏览器自动化 | 不需要 API 也能操作网站 |
| evolink-z-image | AI 图片生成 | 封面配图刚需 |
| youtube-transcript | 视频转文字 | 3 分钟提取 30 分钟精华 |
| pdf | PDF 全能处理 | 读合同、合并拆分、OCR |

### 三个真实踩坑案例

1. **装太多反而乱**：从 19 个精简到 11 个，AI 效率明显提升。够用就停，别陷入配置地狱
2. **有些技能需要额外服务**：RSS 聚合技能需要先部署 RSS 服务器，装了才发现用不了
3. **技能可能重复**：同一功能有 3-5 个不同技能包，让 AI 帮你比较选择

> 💡 Skills 的本质不是"装得多"，是"用得上"。安装前先让 AI 扫一遍安全性，安装后验证能用再保留。

## 第 16 页 · 进阶案例：接入 ClawMate 角色伴侣

> 💬 来源：[ClawMate 开源项目](https://github.com/BytePioneer-AI/clawmate)

### ClawMate 是什么？

ClawMate 是一个为 OpenClaw 打造的**角色伴侣插件**，让你的 AI 助手不再只是冷冰冰的工具，而是一个有温度、有形象、会发自拍的虚拟伙伴。

### 核心能力

| 功能 | 说明 |
|------|------|
| 🕐 时间感知 | 早晨、午休、傍晚、深夜，场景和穿搭随时间自动切换 |
| 📸 情境生图 | 根据对话内容和当前状态生成写实自拍，不是模板套用 |
| 💬 主动发图 | 可配置触发频率，日常聊天中随机发自拍表示关心 |
| 🎭 多角色切换 | 每个角色有独立人设、时间状态和参考图，一键切换 |
| ✏️ 自定义角色 | 通过对话创建自定义角色，AI 引导生成完整角色定义 |
| 🖼️ 多图像服务 | 支持阿里云百炼、火山引擎、fal.ai、OpenAI 兼容接口 |

### 应用场景

- 个人伴侣：日常陪伴、情感交流、生活助手
- 虚拟导师：学习辅导、知识答疑、进度跟踪
- 智能客服：企业服务、品牌形象、客户互动
- 专业顾问：健康管理、心理咨询、职业指导

### 安装方式

```bash
# 一行命令安装（交互式向导）
npx github:BytePioneer-AI/clawmate
```

安装向导会引导你完成：
1. 角色选择（内置角色或自定义）
2. 主动发图配置（触发概率）
3. 图像服务配置（选择图像生成 API）

### 安装后测试

对你的 AI 助手说：

> "发张自拍看看"
> "你现在在干嘛？"
> "帮我创建一个新角色，她是一个喜欢画画的大学生"

### 图像服务配置示例（阿里云百炼）

```json
{
  "defaultProvider": "aliyun",
  "providers": {
    "aliyun": {
      "apiKey": "你的百炼API Key",
      "model": "wan2.6-image"
    }
  }
}
```

### 主动发图配置

```json
{
  "proactiveSelfie": {
    "enabled": true,
    "probability": 0.1
  }
}
```

> `probability` 为每条消息的触发概率，推荐 0.1-0.3。

### 自定义角色

ClawMate 支持通过对话创建自定义角色，AI 会引导你补充角色的性格、背景、穿搭风格等细节，然后自动生成完整的角色定义文件。这与 Day 4 学到的 soul.md 人格定制一脉相承——ClawMate 把人格定制从"文字性格"扩展到了"视觉形象"。

> 💡 ClawMate 展示了 OpenClaw 技能生态的一个有趣方向：AI 不仅能帮你干活，还能成为一个有温度的陪伴者。这也是 AI Agent 从"工具"走向"伙伴"的趋势体现。
