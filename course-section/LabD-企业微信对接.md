## 第 2 页 · 渠道简介

- 🤖 支持创建自建应用机器人

### 适用场景

- 企业内部办公助手

- ✅ Day 2 完成（OpenClaw 已在服务器上运行）

## 第 3 页 · 创建企业微信应用

### 操作步骤

1. 访问 [企业微信管理后台](https://work.weixin.qq.com)，管理员登录
2. "应用管理" → "自建" → "创建应用"
3. 填写应用信息：名称、描述、图标、可见范围
4. 获取凭证：
   - **AgentId**：应用 ID（应用详情页）
   - **Secret**：应用密钥（应用详情页）
   - **CorpId**：企业 ID（"我的企业" → "企业信息"）
5. 配置"接收消息"：
   - 设置服务器 URL
   - 获取 **Token** 和 **EncodingAESKey**
   - 点击"保存"完成验证
6. 开启"API 接收消息"功能

---

## 第 4 页 · 配置 OpenClaw

### 编辑配置文件

```bash
nano openclaw.json
```

### 添加企业微信渠道配置

```json
{
  "channels": {
    "wecom": {
      "enabled": true,
      "corpId": "你的企业ID",
      "agentId": "你的应用AgentId",
      "secret": "你的应用Secret",
      "token": "接收消息的Token",
      "encodingAESKey": "接收消息的EncodingAESKey"
    }
  }
}
```

### 重启 Gateway

```bash
pm2 restart openclaw-gw
```

> ⚠️ 企业微信回调需要服务器有公网 IP 且防火墙已放行端口

## 第 5 页 · 验证消息收发

### 测试步骤

1. 在企业微信手机端或桌面端，找到你创建的自建应用
2. 点击进入应用对话界面
3. 发送测试消息："你好，请介绍一下你自己"
4. 等待 AI 回复
5. 尝试不同类型的消息
6. 确认多轮对话正常

### ✅ 成功标志

- AI 能正常接收消息并回复
- 回复内容符合预期

---

## 第 6 页 · 故障排查速查

| 问题 | 解决方案 |
|------|----------|
| 回调 URL 验证失败 | 确认服务器地址可外网访问；确认 Token 和 AESKey 一致 |
| 应用找不到 | 确认"可见范围"包含你的账号；刷新应用列表 |
| 收不到回复 | 检查日志；确认 Secret 正确；确认接收消息已开启 |
| 回复被截断 | 企业微信单条消息约 2048 字节限制；在 soul.md 中加入"简洁回复" |
